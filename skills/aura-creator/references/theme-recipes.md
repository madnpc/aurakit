# Theme Recipes

Use these recipes as starting points, then adapt from user feedback.

## Ocean

- Base: `#24384f`
- Flow: `#8eefff`
- Accent: `#d7fbff`
- Mood: soft, slow, deep blue, ice-blue water movement.
- Good effects: Static base, Tide flow, low-opacity Comet crest.

## Galaxy

- Base: `#221b3a`
- Flow: `#7c8cff`
- Accent: `#e0e6ff`
- Mood: dark violet, quiet star movement, not rainbow.
- Good effects: Static base, Tide movement, Starry Night or Comet accent.

## Aurora

- Base: `#12342f`
- Flow: `#5ff5c8`
- Accent: `#d8fff2`
- Mood: cyan-green waves, soft northern-light motion.
- Good effects: Static base, Tide movement, slow breathing highlight.

## Tuning Language

- "柔和 / soft": lower brightness and opacity, avoid pure white except small accents.
- "很慢 / very slow": use the lowest speed values and longer durations when present.
- "水流 / flow": prefer Tide.
- "扫过去 / sweep": prefer Comet.
- "不要键盘": remove keyboard from all layer-level device bindings.
